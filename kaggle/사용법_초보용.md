# Kaggle 8B 학습 — 완전 초보용 사용법

> 목표: 로컬(8GB)에서 못 돌리는 **8B 모델**을 Kaggle 무료 GPU로 학습 → 점수 도약 도전.
> 필요한 파일: `KAGGLE_ALL_IN_ONE.py` (이거 하나면 학습+제출 다 됨) + `aux_upload/` 안 CSV 3개.

---

## 📋 준비물 체크
- [ ] Kaggle 계정 (GPU 쓰려면 **전화번호 인증** 필요 — Settings → Phone Verification)
- [ ] 이 폴더의 `KAGGLE_ALL_IN_ONE.py`
- [ ] 이 폴더의 `aux_upload/` 안 CSV 3개

---

## 1단계: 보조 데이터 올리기 (5분, 한 번만)

1. Kaggle 접속 → 왼쪽 메뉴 **Datasets** → **+ New Dataset**
2. `aux_upload/` 폴더 안의 **3개 파일**을 끌어다 놓기:
   - `aug_weights_exp16.csv`
   - `snu_clip_features.csv`
   - `holdout_300.csv`
3. Dataset 이름: 아무거나 (예: `snu-ai-aux`) → **Create**

> 💡 이 파일들은 "어떤 데이터를 얼마나 학습할지" 정하는 우리 노하우입니다. 없어도 돌아가지만 성능이 떨어집니다.

---

## 2단계: 노트북 만들고 코드 붙이기 (5분)

1. Kaggle → 왼쪽 **Code** → **+ New Notebook**
2. 오른쪽 설정 패널에서:
   - **Accelerator** → **GPU T4 x2** 선택 (제일 중요!)
   - **Internet** → **On** (모델 다운로드에 필요)
3. 오른쪽 **Add Input** 버튼 → 두 개 추가:
   - 대회 데이터 (SNU AI Challenge 데이터셋 검색해서 추가)
   - 방금 만든 `snu-ai-aux`
4. 노트북 첫 셀에 **`KAGGLE_ALL_IN_ONE.py` 내용을 통째로 복사 → 붙여넣기**
5. **▶ 실행** (Run) — 또는 상단 **Run All**

> 데이터 경로는 코드가 **자동으로 찾습니다**. 수정할 것 없습니다.
> 실행하면 "✅ 대회 데이터: ..." 가 뜨는지 확인하세요. 안 뜨면 3번(Add Input)을 다시 하세요.

---

## 3단계: 학습 기다리기 (반나절~하루)

- 학습이 시작되면 진행바가 움직입니다. **8B라 느립니다 (10시간 이상 가능).**
- Kaggle은 **한 세션이 최대 12시간**입니다. 12시간에 끊기면:

### ⚠️ 세션이 끊겼을 때 (매우 중요)
Kaggle은 12시간 후 노트북을 멈춥니다. `/kaggle/working`이 날아가지 않게 하려면:

**방법 A (추천, 가장 확실):**
1. 학습 중이든 끝났든, 상단 **Save Version** → **Save & Run All (Commit)** 클릭
2. 이러면 Kaggle이 백그라운드에서 끝까지 돌리고 **결과를 저장**합니다 (최대 12시간)
3. 12시간에도 100% 못 하면: 그 노트북의 **Output을 새 노트북 Input으로 추가** →
   코드의 `CFG["ckpt"]` 부분이 자동으로 이어받습니다 → 다시 Commit
4. "✅ 완주!" 가 나올 때까지 3번 반복

**진행률 확인:** 로그 맨 아래에 `종료: 850/1488 스텝 (57%)` 처럼 나옵니다.
- **99% 미만** = 아직 덜 됨 → 다시 실행(이어감)
- **✅ 완주!** = 다음 단계로

---

## 4단계: 제출 파일 만들기 (1~2시간)

학습이 **✅ 완주!** 되면:

1. 코드 맨 아래쪽에서 이 줄을 찾으세요:
   ```python
   RUN_INFERENCE = False   # ← 학습 완주 후 True로 바꾸고...
   ```
2. `False` → **`True`** 로 바꾸기
3. 다시 실행 → test 819개를 추론해서 `submission.csv`를 만듭니다
4. 오른쪽 **Output** 탭 → `submission.csv` **다운로드**
5. 대회 페이지에 **제출!**

---

## ❓ 자주 나는 문제

| 증상 | 해결 |
|---|---|
| "대회 데이터를 못 찾음" | Add Input에서 대회 데이터셋 추가했는지 확인 |
| 설치가 오래 걸림 | 정상입니다 (2~3분). 인터넷 On 확인 |
| OOM (메모리 부족) 에러 | 코드가 자동으로 그 샘플만 건너뜁니다. "OOM스킵 N" 나오면 정상 |
| 12시간에 끊김 | 3단계 "방법 A" 참고 — 다시 실행하면 이어감 |
| 진행률이 안 오름 | 8B는 원래 느립니다. 로그 시간(h)이 늘면 정상 |
