## 💡 2. 주요 설계 개선점 및 전처리 고도화 (Feedback Integration)

### ① 픽셀 MSE(물리 오차) 한계 극복을 위한 CLIP(의미 오차) 도입
* **기존 한계**: 단순 픽셀 오차(MSE)를 사용해 장면 유사도를 판단할 경우, 카메라 급회전이나 줌인/줌아웃 같은 동작 변화를 장면 전환으로 오판하는 문제가 잦았습니다.
* **해결책**: OpenAI CLIP(ViT-B/32) 모델의 Cosine Distance를 전면 도입하여, 시각적 조명이나 미세 노이즈를 억제하고 이미지 본연의 고차원 의미론적 유사도로 장면 전환을 엄격하게 감지하도록 개선했습니다.

### ② 상대 비율(Ratio) 기법의 무력화 입증 및 절대 기준(0.20) 수립
* **기존 한계**: 비디오 내부 평균 대비 최대값의 비율(Ratio)로 장면을 나누고자 했으나, 4장이 전부 다른 장면(Type 3)은 오차가 고르게 커서 비율이 `1.04`로 급감해 장면 분할에 실패함을 밝혀냈습니다.
* **해결책**: 9,535개 전체 데이터셋의 통계 검증을 거쳐 **절대 임계값 `0.20`** (Z-Score 표준정규분포 기준 **`-0.75`**)을 장면 전환 컷의 최종 경계값으로 확정 및 적용했습니다.

### ③ 엄격 장면 판정 정책 반영 (Strict Cut Algorithm)
* 6개 이미지 쌍 중 Max 오차가 `0.20` 미만인 경우(모두 유사한 경우)는 미세 흔들림 유무와 관계없이 무조건 **장면 전환 0회 (Type 0)**로 묶는 엄격 판정 알고리즘을 설계하여 라벨의 신뢰도를 극대화했습니다.
  * **0회 장면 전환 (정적)**: 2,028개 (21.27%)
  * **1회 장면 전환**: 4,380개 (45.94%)
  * **2회 장면 전환**: 2,038개 (21.37%)
  * **3회 장면 전환**: 1,089개 (11.42%)

### ④ VLM 학습용 소프트 힌트(Soft Hint) Z-Score 연동
* 추출된 CLIP 피처(`Max`, `Mean`)를 QuantileTransformer를 활용해 **Z-Score(평균 0, 표준편차 1)** 형태로 정밀 정규화 완료했습니다.
* 스케일링된 피처 점수는 학습 단계에서 VLM의 입력 프롬프트(Soft Prompt/Hint)에 정교하게 맵핑되어 모델의 순서 추론 정확도를 극대화하는 단서로 활용됩니다.
