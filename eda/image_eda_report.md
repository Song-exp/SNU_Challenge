# 📊 SNU AI Challenge - Image EDA & CLIP Scene Cut Report

이 리포트는 비디오 데이터의 이미지 프레임 전수 탐색적 분석(EDA) 결과 및 의미론적 장면 전환(CLIP) 특징 추출 및 Z-Score 라벨링 설계서입니다.

---

## 1. 이미지 데이터 탐색적 분석 (Image EDA)

전체 훈련 데이터(9,535개 샘플, 총 38,140개 이미지 프레임)를 대상으로 수집한 최종 통계 데이터입니다.

### 1.1 이미지 해상도 분포 (Resolution Distribution)
대부분의 이미지 프레임은 저해상도 형태로 제공되고 있습니다.
* **Top 3 해상도 빈도**:
  1. `640x360` (23,282개 프레임, 약 61.0%)
  2. `480x360` (7,788개 프레임, 약 20.4%)
  3. `320x240` (2,110개 프레임, 약 5.5%)
* **결론**: 전체 이미지의 80% 이상이 `640x360` 이하의 크기를 가지고 있어, 학습 및 추론 시 이미지 입력 크기를 무리하게 키울 필요가 전혀 없습니다.

### 1.2 픽셀 MSE(물리 오차)의 한계와 CLIP(의미 오차)의 도입
이전 연구에서는 단순 픽셀 오차(MSE)만을 사용해 장면 유사도를 판정했으나, 다음과 같은 심각한 오류가 발견되었습니다.
* **픽셀 MSE의 오차**: 카메라가 빠르게 회전하거나(Whip Pan) 급격히 줌인/줌아웃(Zoom)하는 경우, 실제로는 **장면 전환이 없는 동일 씬임에도 불구하고 MSE 값이 10,000을 넘어가며 장면 전환으로 잘못 판단하는 현상**이 빈번하게 관찰되었습니다.
* **의미 오차(CLIP) 도입**: 픽셀 값에 영향을 받지 않고 이미지의 고수준 의미론적 정보를 비교하기 위해 **OpenAI CLIP(ViT-B/32) 모델의 코사인 거리(Cosine Distance)**를 도입했습니다.
  * CLIP 거리는 `0.0` (완벽히 동일한 의미)에서 `1.0` (완벽히 상이한 의미) 사이의 표준 범위로 정규화되어 있어, 조명 변화나 카메라 흔들림 노이즈를 강력하게 억제합니다.

### 1.3 [전수 실증 검증] CLIP 통계적 분포 분석
9,535개 전체 데이터셋을 캐글 GPU로 전수조사하여 얻은 `CLIP_Max`, `CLIP_Mean`, `Ratio (Max/Mean)`의 전체 통계 분포 요약표입니다.

| 통계 지표 | CLIP Max 오차 | CLIP Mean 오차 | Ratio (Max / Mean) |
| :--- | :---: | :---: | :---: |
| **평균 (Mean)** | 0.315 | 0.226 | **1.396** (평균 40% 편차) |
| **표준편차 (Std)** | 0.134 | 0.096 | 0.146 |
| **최솟값 (Min)** | 0.029 | 0.021 | 1.037 |
| **25% 백분위** | 0.214 | 0.155 | 1.287 |
| **중앙값 (50%)** | 0.301 | 0.217 | 1.382 |
| **75% 백분위** | 0.399 | 0.286 | 1.488 |
| **최댓값 (Max)** | 0.842 | 0.639 | 2.150 |

#### ⚠️ 상대적 비율(Ratio) 기법의 무력화 증명
* 동영상 개별 내에서 "평균 대비 몇 % 튀었는가(Ratio)"를 기준으로 장면 전환을 잡을 경우, **장면이 매번 바뀌는 비디오(Type 3)**는 6개 조합이 전부 균일하게 오차가 커서 비율이 `1.04` 수준으로 떨어져 컷을 완전히 놓치게 됩니다.
* 따라서 개별 상대평가 대신, **전체 데이터셋 기준 절대 임계값 `0.20`** (Z-Score 정규분포 기준 **`-0.75`**)을 컷 전환의 통계적 경계선으로 결정했습니다.

### 1.4 엄격 기준에 따른 최종 장면 전환 횟수(0~3회) 분포 (CLIP Max 0.20 기준)
전수 데이터셋을 CLIP Max 오차 `0.20` 엄격 기준(Max < 0.20 이면 0회 전환)으로 격리 및 분류한 최종 통계 분포입니다.

* **🎬 장면 전환 0회 비디오 (정적 미세 행동)**: **2,028개 (21.27%)**
  * 카메라가 완전히 고정된 채 인물의 미세한 움직임만 있는 비디오군입니다. (데이터셋의 `No_ordering=True` 비율인 15.5%에 수렴)
* **🎬 장면 전환 1회 비디오 (2개 씬 분할)**: **4,380개 (45.94%)**
  * 영상 내에 서로 다른 2개의 물리적 공간이 존재하여 경계선이 한 번 존재하는 구조입니다.
* **🎬 장면 전환 2회 비디오 (3개 씬 분할)**: **2,038개 (21.37%)**
  * 3개의 서로 다른 씬이 연속으로 이어지는 비디오 구조입니다.
* **🎬 장면 전환 3회 비디오 (4개 씬 분할)**: **1,089개 (11.42%)**
  * 4개의 프레임이 아예 다른 장소와 구도로 이루어진 극단적인 장면 전환형 비디오입니다.
* **총 비디오 세트 수**: **9,535개 (100.00%)**

## 2. 정규분포(Z-Score) 스케일링 기법

원본 CLIP 거리 값들은 소수점 형태라 VLM(Qwen2-VL) 프롬프트에 소프트 힌트(Soft Hint)로 주입하기에 척도가 일정하지 않습니다. 이를 극복하기 위해 Scikit-learn의 `QuantileTransformer`를 사용하여 표준 정규분포로 스케일링하였습니다.

* **Max_scaled (Z-Score)**: 평균 **`0.00`**, 표준편차 **`1.00`**으로 수렴.
* **의사결정 맵핑 스펙**:
  * `Z-Score < -0.75` ➡️ 장면 전환 0회 (고요한 미세 행동 씬)
  * `Z-Score >= -0.75` ➡️ 장면 전환 1회 이상
  * `Z-Score > 1.00` ➡️ 극단적인 장면 전환 및 역동성 (Type 3)
